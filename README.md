# nano

Personal assistant bot for my Discord channel. Currently under development.

# instructions

1. copy & edit the config file:

    ```shell
    $ cp config.yml.sample config.yml
    $ vim config.yml  # add api keys
    ```

2. run docker compose:

    ```shell
    $ docker-compose up
    ```
